Marly allows you to leverage various language models to transform your data.

These include:
- [GPT 4.0](https://gpt-4.0-docs.com)
- [Azure Open AI](https://azure-open-ai-docs.com)
- [Claude (Coming Soon!)](https://claude-docs.com)
- [Gemini (Coming Soon!)](https://gemini-docs.com)

The following creates a model based on the **Open AI** model
```python model_config.py
model_config = client.configuration.models.create(
	api_key=os.getenv("AZURE_OPENAI_API_KEY"),
	model_name=os.getenv("MODEL"),
	model_type="azure",
	additional_params={
    	"api_version": os.getenv("OPENAI_API_VERSION"),
    	"azure_endpoint": os.getenv("AZURE_OPENAI_ENDPOINT"),
    	"azure_deployment": os.getenv("ENGINE")
	},
	created_by="test-user",
)
my_model = model_config.id
```
