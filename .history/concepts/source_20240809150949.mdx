Sources can include the following:
- Local
- S3
- Gmail
- Postgres
- Dynamo DB

 The following example sets up a local source:

 ```python load_local_document.py
 local_fs_source_config = client.integrations.sources.register_local_fs(
	base_path="/app/example_files"
)

 ```
